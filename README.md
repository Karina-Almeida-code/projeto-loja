## 📌 Sobre o Projeto
O **Vibe Story** é uma landing page desenvolvida com foco em apresentação de uma marca de moda urbana.  
O projeto destaca uma nova coleção com um design moderno e visual impactante, simulando uma loja online.

## ✨ Funcionalidades
- 🛍️ Apresentação de coleção (Nova Coleção Street)  
- 🎯 Botão de chamada para ação (Ver Produtos)  
- 🎨 Layout moderno inspirado em e-commerce  
- 📱 Design responsivo  
- 🧱 Estrutura simples e organizada  

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  

## 🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de:
- Praticar criação de landing pages  
- Trabalhar design voltado para e-commerce  
- Melhorar habilidades em HTML e CSS  
- Criar interfaces modernas para portfólio
